# Spotify Top Songs Analysis

Análise exploratória de um dataset do Kaggle contendo as 100 músicas mais ouvidas do Spotify até a data de publicação no site. O projeto explora métricas como streams, gênero, país de origem e características musicais, com o objetivo de extrair insights relevantes usando as bibliotecas necessárias no Python.


# Principais Análises

- Top artistas por número total de streams  
- Distribuição de músicas explícitas vs não explícitas  
- Análise de streams por país de origem  
- Exploração de características musicais (danceability, energy, etc.)


#  Exemplos de Visualizações

- Gráfico de barras com os artistas mais populares  
- Gráfico de pizza com proporções de categorias  
- Outras análises exploratórias com base nos dados  


# Tecnologias Utilizadas

- Python  
- Pandas  
- Matplotlib
